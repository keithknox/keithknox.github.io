Vineyard Itinerary

Creating a map of vineyards in the Shenandoah Valley was my third project for General Assembly. Using both the Google Maps and Yelp APIs, the goal of the site was to allow users to add entries from the Yelp API to a Google map and create a custom wine tour. Though the Google Maps portion proved to be a little difficult with the deadline given, I was able get a deeper understanding of the Yelp API using the "bounding box" function since the Shenandoah Valley is not a metropolitan area. This project definitely aided in my overall comprehension of APIs.

Keith's Cookbook

My final project for General Assembly, a Ruby on Rails app that enables users to upload and comment on recipes. This was my first time implementing AJAX and jQuery on a project as well as experimenting with the Materialize CSS framework. The AJAX functionality can be seen when you add comments to the recipes without a page reload. Though not as robust as other websites, I am very proud of this app and shows how far I have come since starting General Assembly in March.
