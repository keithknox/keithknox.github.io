About Me

I am a former payroll accountant looking to make the leap into full stack web development. I graduated General Assembly's Web Development Immersive in June 2015.

Projects

Invisible Illness

This project was a group effort in my General Assembly class that combined 3 developers and one designer from the User Experience class. The focus of the project was to tell a story and our designer proposed the idea of creating a forum for people with invisible illnesses. Users could personal stories, find links for more information and even contrbute to a charity if they wished to do so. My role was taking the designers proofs and creating the front-end style for the page, which was tricky because we hadn't learned any Javascript yet. All the animations and layout were done in CSS over the course of two days. I am pretty proud of this project because we worked well as a team and met the "client's" expectations with what little front-end skill we had at that time.

Vineyard Itinerary

This was my third project for General Assembly and the idea was to create a map of vineyards in the Shenandoah Valley and choose to add them to an itinerary so that they could create their own wine tour. It was to incorporate the Google Maps API and the vineyards would come from the Yelp API. I got the Yelp API working, but didn't have much luck plotting the points on Google Maps. The one thing I was proud of was being able to seed my database with the Yelp data by using a "bounding box" function. Since the Shenandoah Valley is not a metropolitan area, I had to define where to look for these vineyards and I'm proud of the results.

Keith's Cookbook

This was my final project for General Assembly. It is a Ruby on Rails app that allows users to upload recipes and comment on them. This was the first time I was implementing AJAX and jQuery on a project as well as trying out the Materialize CSS framework. The AJAX calls can be seen when you add comments to the recipes without a page reload. It may be simple, but I'm still very proud of this app since it has shown how far I have come since starting General Assembly in March.
